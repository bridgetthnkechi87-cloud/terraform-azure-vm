# Azure VM Deployment with Terraform

## Overview

This project provisions an Azure Virtual Machine using Terraform.

---

## What I Built

* Resource Group
* Virtual Network (VNet)
* Subnet
* Public IP
* Network Interface
* Linux Virtual Machine

---

## Tools

* Terraform
* Azure
* Linux

---

## Deployment

```bash
terraform init
terraform apply
```

Successfully deployed and connected to an Azure VM using Infrastructure as Code.
